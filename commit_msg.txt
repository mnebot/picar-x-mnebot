Configura SonarCloud per analitzar només la carpeta gpt_examples

Afegeix configuració de SonarCloud per centrar l'anàlisi en la carpeta
gpt_examples amb exclusiones lògiques per fitxers no rellevants.

Canvis principals:

1. Crea fitxer sonar-project.properties:
   - Configura projectKey i organization
   - Defineix sources només a gpt_examples
   - Afegeix exclusiones per documentació (*.md, *.txt)
   - Afegeix exclusiones per imatges (*.png, *.jpg)
   - Exclou cache de Python (__pycache__, *.pyc)
   - Configura Python 3.11 i encoding UTF-8

2. Actualitza workflow de GitHub Actions:
   - Elimina paràmetres duplicats dels args
   - Workflow ara llegeix tota la configuració del fitxer
     sonar-project.properties (font única de veritat)
   - Afegeix comentaris explicatius

Aquesta configuració assegura que SonarCloud només analitzi el codi
Python rellevant de gpt_examples, excloent fitxers de documentació,
imatges i cache que no aporten valor a l'anàlisi de qualitat del codi.
